Learning Activities & Resources

What is a Child Theme?

Week 5 activity included learning and developing child themes in a WordPress. During all the development of websites for practical session, I have edited the parent theme itself, removing most of the theme's styles, which was not quite a good way to make changes. This is why child theme is important as it allows us to change small aspects of site's appearance, preserving theme's functionality. It is an extension of a parent theme.

How to build a child theme

Before starting to create child theme, it is significant to navigate/know the location of file manager of the WordPress site. At first, I worked on creating child theme on my live server WordPress site developed previously. However, it was quite a hurdle to locate file manager and even after finding it, I realized creating ZIP file from the live server site will be a hectic task later. Then, I started to create my mytheme child locally from my local environment.
For creating a child theme, it required us to navigate to ..wp-content/themes and create new folder with a style.css and functions.php files inside that folder. Knowing this simple creation of folder and file helped me to successfully build child themes for the practical 5.

Challenges faced while working on practical

Regardless of how many times I have customized mytheme child's css file, it was hard to get it implemented on my site. What's amusing was that even after activating my custom child theme and customizing from the WordPress customization, the style.css of my child theme did not get updated. It only consisted of the commands I made. I also tried using the inspection developers tool to find the exact selector of elements but solutions couldn't be obtained. Nontheless, I understood the working of child theme and how it should be utilized.
