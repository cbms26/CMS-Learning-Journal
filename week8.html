Learning Activities

Learning Experiment topic: Consistent Practice with Iterative Feedback Enhance CMS Proficiency

Theory

The theory of deliberate practice (Source: https://jamesclear.com/deliberate-practice-theory) suggests that a consistent and focused practice, when integrated with feedback loop for improvement, helps in expertising complex skills. This theory hypothesize that a skill practice must be intentional involving reflective feedback loop rather than performing a task repetitively until mastered.

Hypothesis

Using the theory of deliberate practice to solve CMS practical and assessments, in order to have significant improvements in practical skills, understanding and the ability to fix any issues.

Methodology

  1. Preparation
    • Conduct self-assessment such as quiz or practical task or assignment to measure my existing CMS skills.
    • Choose a CMS platform (for e.g. WordPress), then develop any website project. This should help serve me as the basis for iterative development and practice.
  2. Practice Session
    • Establish a schedule to achieve consistent practice. For instance, I have taken one session of 4-6 hours each week to complete my CMS practical tasks. Each session or practical task required me to focus on praticular aspect of CMS such as plugin installation, child themes and so on.
    • Develop the project iteratively, starting with basic setup and then adding features. Maintain a document of my progress, challenges faced and solutions required for each issues. For example, working on CMS assignment 1: Website Development required all these to successfully complete.
  3. Feedback Loop
    • After every session, self-reflect on the work similar to maintaining Learning Journal each week of the CMS class session.
    • Get peers feedback of the progress made to identify if there is any improvement needed.

Test

  1. Post-Practice Tasks
    • Similar to weekly practical task of CMS subject, performing a set of tasks to those learnt concepts during each learning session. This helps measure the improvment in understanding and skills
  2. Project Evaluation
    • Conducting detailed review of the completed project.
  3. Skill Recall
    • After a break of a week, perform another new task to uptake new issues. This will help in retaiing the skills learned.

Results

  • Assessment Scores
  • Project Quality
  • Retention Test Performance
