Getting started with Joomla

I skimmed through the Learning Resources Video for week 2 in LinkedIn which explained on content management system and using of WordPress. As for the practical, Joomla CMS was entirely new to me. I went through various sites and youtube videos (mainly WebTech with Brett), helping me understand how to install Joomla, what it is, and how to use it for developing a website.

Working with Joomla

Frankly, Joomla installation was not an easy task. However, after successfully installing Joomla, it took some time to get fond with the features and functionality included. I found out that we can also, do html coding and created custom css to override the default css styling of Joomla. Hence, I almost worked more on using html and css skills inside the Joomla article content, where I finally created my website 'CODEBOTs' using Joomla version 5.

Exporting Joomla files to Github repository for web-hosting

Initially, I could not upload my Joomla files to GitHub repo. I learned that backing up or exporting Joomla files allows me to do that. Akeeba Backup was a useful Joomla Extension Directory, which allowed me to create full backup of my Joomla site. After this process, I directly committed my Joomla files to the github repo. I generated the website link, where it only displayed 'Error 404'. Just like I hoped, that it won't be an easy task to shift or get a live server. Upon researching the error, I learned that Github provides the web-hosting for static website only. Any CMS websites created are dynamic. Then, I converted my dynamic website to static by some free third-party software (called HTTrack Website), then pushed it to Github which resolved the error.
